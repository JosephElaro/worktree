The **W0rkTree Admin Panel** (`worktree-admin`) provides a web-based management interface for monitoring and controlling W0rkTree server instances. It includes a RESTful API, real-time dashboards, and server management tools.

<Callout type="info">
The Admin Panel also ships with a TUI (Terminal User Interface) dashboard for quick server inspection from the terminal.
</Callout>

## Overview

The Admin Panel provides:

- **Server monitoring** — real-time health, metrics, and performance dashboards
- **Repository management** — create, configure, and manage hosted repositories
- **User management** — manage users, teams, and organizations
- **Maintenance tools** — garbage collection, backup, and recovery
- **REST API** — full programmatic control over all admin operations
